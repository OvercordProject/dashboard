# Overcord

[![MIT License](https://img.shields.io/apm/l/atomic-design-ui.svg?)]()

Overcord is a discord server project based on the famous game Overwatch.

## Self host

Clone this repos

```shell
npm install
```
or
```shell
pnpm install
```

Edit .env.example => .env with your credentials

```sh
npm run dev
```
